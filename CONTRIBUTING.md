# Contributing

Welcome to GEOStudio Core! If you are interested in contributing to the [GEOStudio Core code repo](README.md) then checkout the [Contribution Process](./docs/contributing_process.md) and the [Code of Conduct](CODE_OF_CONDUCT.md).

The [GEOStudio Core code repo](README.md) contains information on how the community is organized and other information that is pertinent to contributing.

## Getting Started

It's encouraged that you look under the [Issues](https://github.com/terrastackai/geospatial-studio-core/issues) tab for contribution opportunities.
