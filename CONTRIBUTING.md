# Contributing

Welcome to GEOStudio Pipelines! If you are interested in contributing to the [GEOStudio Pipelines code repo](README.md) then checkout the [README](./README.md) and the [Code of Conduct](CODE_OF_CONDUCT.md).

The [GEOStudio Pipelines code repo](README.md) contains information on how the community is organized and other information that is pertinent to contributing.

## Getting Started

It's encouraged that you look under the [Issues](https://github.com/terrastackai/geospatial-studio-core/issues) tab for contribution opportunities.
